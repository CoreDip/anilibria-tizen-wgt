## 📺 Установка

### Вариант 1 — Developer Mode (Tizen)
1. На TV: **Настройки → Общие → Системный менеджер → Режим разработчика** → ввести IP ПК
2. Перезагрузить TV
3. В Tizen Studio: `Run As → Tizen Web Application` или установить `.wgt` через `Package Manager`

### Вариант 2 — MediaStationX
1. Открыть MediaStationX на TV
2. Добавить URL: `aniliberty.netlify.app`

### Вариант 3 — ПК / Планшет / Браузер / Телефон
Просто открыть [aniliberty.netlify.app](https://aniliberty.netlify.app) — адаптивный интерфейс подстраивается под устройство автоматически.

## ✨ Возможности

**🏠 Главная**
- Ряды: новые серии, онгоинги, популярное, топ по рейтингу, по жанрам
- Прогресс-бар просмотра прямо на карточке
- Ленивая подгрузка рядов при скролле
- Контекстное меню на карточке (долгое нажатие / ПКМ)

**🔍 Каталог**
- Поиск по названию
- Фильтры: жанр, год, сезон, тип, статус выхода, статус озвучки, возрастная отметка, сортировка
- На мобильном — фильтры в выезжающей панели снизу
- Бесконечная подгрузка результатов
- Восстановление позиции скролла при возврате из страницы аниме

**📅 Расписание**
- Выход серий по дням недели с постерами
- На мобильном — вертикальный список с горизонтальной прокруткой по дням

**📂 Коллекции**
- Вкладки: Избранное, Смотрю, Запланировано, Просмотрено, Брошено, История
- Значки коллекции на карточках — виден статус аниме (❤ / 👁 / 📅 / ✔ / ✖) без открытия страницы
- Прогресс-бары отображаются в коллекциях сразу, без перехода на каждое аниме
- При добавлении в «Просмотрено» все серии автоматически отмечаются просмотренными
- Добавление и удаление через контекстное меню или страницу аниме
- Удаление из истории, сброс прогресса

**▶️ Плеер**
- HLS-стриминг (нативный на Tizen, hls.js на остальных)
- Выбор качества: 480p / 720p / 1080p
- Кнопка «⏩ Пропустить опенинг» — появляется автоматически, пропускает одним нажатием
- Кнопка «⏭ Следующая серия» — появляется во время эндинга, не нужно ждать титры
- Ускоренная перемотка зажатой стрелкой (5с → 10с → 30с)
- Скраббинг по таймлайну
- Автопереход к следующей серии
- Переключение режима вписывания: по рамке / заполнить экран
- Кнопка полноэкранного режима (ПК и мобильный)
- Кнопка «Назад» в верхнем левом углу

**💾 Прогресс и синхронизация**
- Автосохранение позиции каждые 8 секунд
- Полная двусторонняя синхронизация с аккаунтом AniLibria
- Миграция локального прогресса в облако при входе
- Фоновый синк коллекций и истории каждые 2 минуты
- Кнопка «Продолжить» с точного места остановки
- Сброс прогресса удаляет данные сразу и в облаке
- Долгое нажатие на кнопку серии — снять отметку о просмотре (мышь, тач, пульт ДУ)

**👤 Аккаунт**
- Вход по логину и паролю
- Синхронизация коллекций и истории с сервером
- Профиль с кнопкой выхода

**🎮 Пульт ДУ (Samsung Tizen)**
- Полная навигация стрелками без мыши
- Кнопка Back — шаг назад по иерархии
- Двойное нажатие Back — меню выхода из приложения
- Долгое нажатие OK на карточке — контекстное меню
- Долгое нажатие OK на серии — снять отметку о просмотре
- Поддержка цветных кнопок (каталог, расписание, коллекции)

**🖱️ ПК (мышь)**
- Наведение, клик, правая кнопка мыши — контекстное меню
- Стрелки на рядах для прокрутки карточек
- Кнопка полноэкранного режима

**📱 Мобильный (Android / iOS)**
- Адаптивный интерфейс под экраны телефонов
- Нижняя навигация: Главная / Каталог / Расписание / Коллекции
- Фильтры каталога в выезжающей панели снизу (свайп вниз или кнопка ☰)
- Долгий тап на карточке — контекстное меню
- Свайп вправо на странице аниме — назад

**⚡ Производительность**
- Оптимизировано для Tizen: без тяжёлых анимаций и blur-эффектов
- Виртуализация карточек (15 в DOM на ряд)
- Кэширование данных в памяти
- Масштаб интерфейса +30% на Tizen для комфортного просмотра с дивана

## 🛠️ Разработка

```bash
git clone https://github.com/CoreDip/anilibria-tizen-wgt.git
# Открыть index.html в браузере для разработки
# Или импортировать папку tizen/ в Tizen Studio как Existing Project
```

```
anilibria-tizen-wgt/
├── index.html           ← всё приложение (single-file)
├── config.xml           ← Tizen манифест
├── icon.png
├── js/
│   └── main.js          ← регистрация кнопок пульта
├── css/
│   └── style.css
├── msx/
│   └── start.json       ← конфиг для MediaStationX
└── .github/workflows/
    └── build.yml        ← автосборка .wgt + GitHub Pages
```
